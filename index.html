Quotes Endpoints

  • View All Quotes - Retrieves all quotes.
  • View Quote by ID (ID 10) - Retrieves a single quote by ID.
  • View Quotes by Author (Author ID 5) - Retrieves quotes by a specific author.
  • View Quotes by Category (Category ID 4) - Retrieves quotes by a specific category.
  • View Quotes by Author and Category (Author with ID 5, Category with ID 4) - Retrieves quotes by both author and category.
  • POST /api/quotes/ - Create a new quote.
  • PUT /api/quotes/ - Update an existing quote.
  • DELETE /api/quotes/?id=10 - Delete a quote.

Authors Endpoints

  • View All Authors - Retrieves all authors.
  • View Author by ID (ID 5) - Retrieves a single author by ID.
  • POST /api/authors/ - Create a new author.
  • PUT /api/authors/ - Update an existing author.
  • DELETE /api/authors/?id=5 - Delete an author.

Categories Endpoints

  • View All Categories - Retrieves all categories.
  • View Category by ID (ID 4) - Retrieves a single category by ID.
  • POST /api/categories/ - Create a new category.
  • PUT /api/categories/ - Update an existing category.
  • DELETE /api/categories/?id=4 - Delete a category.
