Welcome to the Quotes REST API

Explore the following endpoints to interact with quotes, authors, and categories:

Quotes Endpoints

  • View All Quotes - Retrieves all quotes.
  • View Quote by ID (e.g., ID 10) - Retrieves a single quote by ID.
  • View Quotes by Author (e.g., Author ID 5) - Retrieves quotes by a specific author.
  • View Quotes by Category (e.g., Category ID 4) - Retrieves quotes by a specific category.
  • View Quotes by Author and Category (e.g., Author 5, Category 4) - Retrieves quotes by both author and category.
  • POST /api/quotes/ - Create a new quote (send JSON data).
  • PUT /api/quotes/ - Update an existing quote (send JSON data).
  • DELETE /api/quotes/?id=10 - Delete a quote (specify ID).

Authors Endpoints

  • View All Authors - Retrieves all authors.
  • View Author by ID (e.g., ID 5) - Retrieves a single author by ID.
  • POST /api/authors/ - Create a new author (send JSON data).
  • PUT /api/authors/ - Update an existing author (send JSON data).
  • DELETE /api/authors/?id=5 - Delete an author (specify ID).

Categories Endpoints

  • View All Categories - Retrieves all categories.
  • View Category by ID (e.g., ID 4) - Retrieves a single category by ID.
  • POST /api/categories/ - Create a new category (send JSON data).
  • PUT /api/categories/ - Update an existing category (send JSON data).
  • DELETE /api/categories/?id=4 - Delete a category (specify ID).

API Usage Notes:
- For POST and PUT requests, send JSON data with the required fields (quote, author_id, category_id, author, category).
- Replace example IDs with actual values.
